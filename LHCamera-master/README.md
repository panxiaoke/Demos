##自定义水印相机

*介绍
前段时间公司的项目需要做自己的自定义相机和照片水印。抽空记一下笔记。文末有Demo

实现了相机的自定义 和水印、滤镜相关功能。image处理方式都是用类别的方式。

- •闪光灯的自定义开关
- •切换前后摄像头的开关
- •相机的缩放
- •相机的点击聚焦
- •使用陀螺仪矫正相片的横竖屏拍照
- •拍照照片的截取处理
- •添加水印制作和滤镜功能
- •Image的剪裁、缩放、旋转等处理方法
- •水印的交互处理 （拖动、删除、放大、虚框隐藏预览）//LHStickerView
